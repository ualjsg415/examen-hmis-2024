### Apartado 1
Para fusionar la rama apartado1 sobre main de manera que fuera necesario 
un merge non-fast forward, primero añadí un commit extra en main después 
de crear la rama apartado1, de forma que ambas ramas divergieran. 
Luego ejecuté el comando:

git merge --no-ff apartado1 -m "Merge non-fast forward de apartado1 en main"

El flag --no-ff fuerza la creación de un commit de fusión aunque no sea 
estrictamente necesario, y al haber divergido las ramas se produjo 
un merge non-fast forward real.

### Apartado 2
Para deshacer el último commit de forma que desapareciera de la historia usé:

git reset --hard HEAD~1

Para deshacer los cambios del último commit sin que desapareciera de la 
historia (porque ya estaba publicado en GitHub) usé:

git revert HEAD --no-edit
